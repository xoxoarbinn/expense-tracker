# Expense Tracker

A simple expense tracking app built with HTML, CSS, and JavaScript.

## Features
- Add expenses with name, amount, and category
- Delete expenses
- Running total calculation
- Responsive two-column layout

## Tech Stack
- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript (DOM manipulation)



## What I Learned
- DOM manipulation with createElement, appendChild, remove
- Event listeners for click and keyboard events
- JavaScript scope and state management
- Flexbox layout design
